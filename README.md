# 💰 Calculadora Sueldo Líquido Chile 2026

![Licencia MIT](https://img.shields.io/badge/licencia-MIT-blue)
![Vanilla JS](https://img.shields.io/badge/JavaScript-ES6+-yellow)
![Tailwind](https://img.shields.io/badge/Tailwind-CSS-06B6D4)

Aplicación web **minimalista y mobile-first** para calcular el sueldo líquido mensual en Chile, cumpliendo con la legislación vigente a **2026**. Incluye desglose detallado de AFP, salud (Fonasa/Isapre), seguro de cesantía e impuesto único de segunda categoría.

🔗 **[Ver Demo en GitHub Pages](https://tuusuario.github.io/calculadora-sueldo-liquido-chile/)**

![Preview Light/Dark](https://via.placeholder.com/800x400?text=Captura+Modo+Claro+Oscuro)

---

## ✨ Características Principales

| Funcionalidad | Descripción |
|---------------|-------------|
| 📱 **Mobile‑First** | Interfaz adaptada a smartphones, diseño limpio y espaciado óptimo. |
| 🌗 **Modo Oscuro / Claro** | Detecta preferencia del sistema y permite cambio manual. |
| 💵 **Formato CLP Automático** | Input de sueldo bruto con separadores de miles. |
| 🏦 **AFP Actualizadas** | Lista desplegable con comisiones 2026 + opción de porcentaje personalizado. |
| 🏥 **Salud Flexible** | Fonasa (7%) o Isapre con plan pactado en UF. |
| 📊 **Desglose Detallado** | AFP, Salud, Cesantía, Base Tributable, Impuesto y Líquido final. |
| ⚙️ **Variables Configurables** | UF, UTM, Tope Imponible y tramos de impuesto editables en `script.js`. |
